# react-spotify-player

## Player to control your spotify web application

## Features

- Control Your Songs
- Control on which device to stream the music
- Control Volume
- Shortcut to create Song Radio
- Like Songs
- See your current playlist
- Shuffle and repeat mode

## Install

`yarn install`

## Run

`yarn start`

## Build

`yarn build`
